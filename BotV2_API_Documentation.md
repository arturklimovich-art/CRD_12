# ДОКУМЕНТАЦИЯ API BOT V2
========================

## Submit-TZ
**Назначение:** Подача технического задания
**Параметры:**
- -Text (string, обязательный) - Текстовое описание ТЗ
**Возвращает:** Объект с TraceId, PlanId, ComplexityScore

## New-TZPlan  
**Назначение:** Создание плана выполнения
**Параметры:**
- -Text (string, обязательный) - Текст ТЗ для анализа
- -Complexity (int, обязательный) - Оценка сложности (1-100)
**Возвращает:** Объект с планом выполнения

## Queue-EngineerJobs
**Назначение:** Постановка задач в очередь
**Параметры:**
- -PlanId (string, обязательный) - ID плана для выполнения
**Возвращает:** Объект с количеством поставленных задач

## Watch-Plan
**Назначение:** Мониторинг выполнения плана
**Параметры:**
- -PlanId (string, обязательный) - ID плана для мониторинга
**Возвращает:** Объект со статусом выполнения

## Refine-Task
**Назначение:** Уточнение неудачной задачи
**Параметры:** Требует уточнения

## Attach-CandidatePatch
**Назначение:** Прикрепление кандидата-патча
**Параметры:** Требует уточнения

## Get-BotModuleInfo
**Назначение:** Получение информации о модуле
**Параметры:** Нет
**Возвращает:** Объект с информацией о модуле
