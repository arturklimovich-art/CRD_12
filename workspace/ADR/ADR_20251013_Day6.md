# ADR Day 6 — Самолечение и снапшоты

## Контекст
Необходим механизм самовосстановления системы CRD12 после ошибок и сбоев.

## Решение
- Добавлены таблицы \snapshots\, \ecovery_logs\.
- Реализован класс \SelfHealingSystem\ для создания и восстановления снапшотов.
- Добавлены API эндпоинты \/system/snapshot\ и \/system/restore/{id}\.
- Обновлён PASSPORT.md (раздел «Самолечение»).
- Созданы артефакты SPEC/PLAN/CHECKLIST/RESULT.

## Последствия
Система стала «неубиваемой»: есть 3 уровня восстановления — быстрый рестарт, восстановление из снапшота и полный rebuild.

## Статус
Принято.
