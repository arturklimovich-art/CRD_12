# ADR_20251013_Day7 — Stabilize runtime

Context: compose помечал engineer_b_api как unhealthy из-за небезопасного healthcheck.
Decision: валидный healthcheck (python urllib), depends_on: service_healthy для bot.
Consequences: стабильный health; бот стартует после API; риски низкие; БД/секреты не тронуты.
Evidence: /ready=200; docker inspect=healthy; smoke/load тесты зелёные.
